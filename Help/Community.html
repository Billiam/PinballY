Community Links

Here are some places on the Web where you can connect with other PinballY people.

  • The Virtual Pinball Cabinets forum on vpforums. This is a busy forum for everything related to building pin cabs, so it's a good fit for PinballY topics, given that PinballY is designed especially for pin cabs. I'm usually pretty active here, so it's a good place to reach me.
  • My "Call for testing" thread in the pin cabs forum above. This is where I originally announced the program, and it became the program's de facto support thread for questions and bug reports. It's ridiculously lengthy at this point, but I monitor it and try to follow up when appropriate, so it's a good place to post questions.
  • PinballY group on Facebook, created by cherlihy71.
  • PinballY issues page on github. This is the github bug tracker for the project. It can be used for bug reports and enhancement requests. I monitor this regularly.

Please pass along any other links that I should add here!
