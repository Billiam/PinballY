Button Sounds

You can customize the sound effects played when navigation buttons are pressed by supplying your own .WAV files with the sounds you'd like to play.

Folder location

Startup videos go in the Button Sounds folder within your main Media folder. See Media folder location for more on where that folder is located.

File types

Button sounds must be .WAV files.

Individual button file names

To customize the sound effect for a particular button, just place a file in the Button Sounds folder with the name matching the button, as shown below.

FilenameButton Description
AddCredit.wav Coin-in sound when a full credit is added. See Coin Options for more on how the coin-in buttons are handled, and the distinction between a coin-in event that adds a partial credit and one that adds a full credit.
CoinIn.wav Coin-in sound when a partial credit is added
Deselect.wav The "Exit" button, when closing a menu or popup
Launch.wav Played each time a game is launched
Next.wav The "Next" button, moving right through the wheel or to the next menu item
Prev.wav The "Previous" button, moving left through the wheel or to the previous men item
Select.wav The "Select" button: opening a menu, selecting a menu command, opening a popup

Disabling an individual button sound

If you don't want to hear a sound effect at all when you press a particular button, you can simply use a WAV file with a recording of an interval of silence for that button's sound file.

For your convenience, we provide a pre-made silent WAV file for this purpose. You'll find the file in the Assets folder in your PinballY program folder, under the name Silence.wav. Simply copy this file to the Media\Button Sounds folder as explained above, and rename it to match the button sound you want to suppress. Repeat that for each button you want to silence.

Muting all button sounds

If you just want to disable all of the button sounds across the board, you can use the "Mute Buttons" command in the Operator Menu.
