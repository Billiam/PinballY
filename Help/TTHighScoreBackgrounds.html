"TT" High Score Backgrounds

The "TT" (typewriter) high score style is designed for older games from the electromechanical era. Games from that era didn't usually have any sort of high-score memory built into the machines themselves, since it would have been too complex and bulky to build that kind of circuitry using mechanical score reels. So if anyone kept track of high scores at all, it was by writing down them down on bits of paper and sticking them to the machines. The "TT" style is designed to re-create that effect by showing typewriter text on an index-card background.

If you prefer to replace the default index-card background with something else, you can provide your own custom image file as described below.

You can also customize the font and text color used for the TT-style high score display cards. See the TT High Scores item in Font and Color Options.

Folder location

A custom background image for the "TT" high score style goes in the Images folder within your main Media folder.

See Media folder location for more on where that folder is located.

File name

The filename for a custom "TT" background image is TT High Score Background.ext, where .ext is one of the standard image file extensions.

File types

The extension can be any of the standard image file suffixes: .png, .jpg, .jpeg, or .gif.
