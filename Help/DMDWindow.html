dmdWindow Object

dmdWindow is a pre-defined PinballY object representing the DMD window in the user interface. The DMD window (for "Dot Matrix Display") is the window that simulates the plasma score display on 1990s+ pinball machines. It's also used to emulate the alphanumeric display on 1908s games.

Methods and properties

This object provides the common window methods and properties. It doesn't currently have any additional properties or methods beyond the common set.
