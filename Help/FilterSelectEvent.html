FilterSelectEvent

This event class is fired when a new filter is selected for the wheel UI.

This event has only one subtype:

  • filterselect: Fires when a new filter is selected. This event fires before the filter is put into effect, so you can cancel the event to prevent the filter change.

The event target for this event type is the gameList object.

Properties

This event type has all of the standard event properties and methods (see the Event class). In addition, it has the following:

  • id: A string containing the ID of the new filter. You can pass this to gameList.getFilterInfo to retrieve more information on the filter.
