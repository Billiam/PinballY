WheelModeEvent

This event class is fired whenever the program returns to the base "Wheel" user interface, after being in another modal UI state (showing a popup, showing a menu, in attract mode, running a game).

This event has only one subtype:

  • wheelmode: Fires when returning to the regular wheel UI. This event fires after the fact, so it's not cancelable.

Properties

This event type has all of the standard event properties and methods (see the Event class). It has no additional properties beyond the standard set.
